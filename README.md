# Bantu Image Fixer

这是一个 WordPress 插件，旨在禁止 WordPress 生成缩略图和裁剪图片，并提供工具将现有文章的特色图片替换回原图（通过清除缩略图元数据）。

![插件预览](https://www.sdczz.com/wp-content/uploads/2026/02/20260210082316_d4e06.png)

## 功能特点

1. **禁止生成缩略图**：上传图片时，禁止生成任何尺寸的中间图片（thumbnail, medium, large 等）。
2. **禁止生成超大尺寸缩放**：禁止 WordPress 5.3+ 默认生成的 `-scaled` 图片。
3. **修复现有图片**：提供一个后台工具，扫描所有已发布的文章，并强制将其“特色图片”重置为**原图**。

## 安装方法

1. 下载插件压缩包或直接上传文件到 `wp-content/plugins/` 目录。
2. 在 WordPress 后台点击 **插件 -> 已安装插件**，找到 **Bantu Image Fixer** 并激活。

## 使用说明

### 1. 自动生效功能
插件激活后，新上传的图片将不再生成任何缩略图或裁剪版本，直接使用原图。

### 2. 修复现有图片
如果您的网站已有大量文章使用了缩略图，您可以使用本插件提供的工具将其批量替换为原图。

1. 进入 WordPress 后台，点击 **设置 -> Bantu 图片修复**。
2. 阅读页面上的说明和警告。
3. 点击 **开始扫描并替换为原图** 按钮。

**原理说明**：
程序会查找文章的特色图片，从数据库中删除其“缩略图尺寸”记录。这会迫使 WordPress 在前端调用该图片时，因为找不到缩略图记录而直接使用原图。
（例如：调用 `image-768x432.webp` 会变为直接调用 `image.webp`）

## ⚠️ 重要警告

- **操作不可逆**：使用“修复现有图片”功能会修改数据库中的图片元数据（删除 `sizes` 信息）。如果不满意结果，您可能需要使用“重新生成缩略图（Regenerate Thumbnails）”类插件来恢复。
- **备份建议**：强烈建议在执行批量修复操作前 **备份您的数据库**。
